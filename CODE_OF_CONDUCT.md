# Code of Conduct

## Our Pledge
我们致力于为所有贡献者和用户提供一个开放、友好、无骚扰的社区。

## Our Standards
- 友好、尊重的沟通
- 接受有建设性的批评
- 关注社区整体利益
- 遵守法律与隐私保护

不可接受的行为包括：人身攻击、辱骂、骚扰、公开或私下披露他人敏感信息、任何违法内容。

## Enforcement Responsibilities
维护者负责解释和执行本准则，可对不当行为采取临时或永久的限制措施。

## Scope
本准则适用于项目空间、GitHub 社区互动，以及代表项目的公开场合。

## Enforcement
- 首选途径：使用 GitHub 的 “Report abuse” 或本仓库的私密安全报告渠道（见 `SECURITY.md`）。
- 也可私信维护者（如果已提供联系人），尽量附上时间、地点、截图或链接。
